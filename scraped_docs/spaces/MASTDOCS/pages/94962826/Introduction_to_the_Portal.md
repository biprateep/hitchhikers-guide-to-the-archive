<!-- CHUNK 3 START -->
<!-- Metadata: {"Header 1": "Important Things to Know", "Header 2": "Browser Cookies"} -->
The Portal site uses browser cookies in order to monitor feature usage, track user preferences, and provide authentication for some services.
Consent for Cookies
By using this site you consent to the use of cookies for these purposes.

<!-- CHUNK 3 END -->

<!-- CHUNK 4 START -->
<!-- Metadata: {"Header 1": "Important Things to Know", "Header 2": "Browser Pop-ups"} -->
The Portal uses browser pop-ups for some functions. This requires that users disable pop-up blockers for the domain: <https://mast.stsci.edu/> in order to download data. Check the documentation for your browser to determine how this is done, and be alert for notifications from the browser that popups are being blocked. In many cases it simply requires giving approval for popups when you are asked.
Failing to disable the browser pop-up blocker for the Portal will result in the silent failure of some functionality.

<!-- CHUNK 4 END -->

<!-- CHUNK 5 START -->
<!-- Metadata: {"Header 1": "Important Things to Know", "Header 2": "Browser Compatibility"} -->
The MAST Portal works well with recent versions of the Chrome, Firefox, and Safari browsers. However, there are **known incompatibilities** with the **Internet Explorer** and **Edge** browsers.

<!-- CHUNK 5 END -->

<!-- CHUNK 6 START -->
<!-- Metadata: {"Header 1": "Important Things to Know", "Header 2": "Login"} -->
Searches for data may be performed anonymously, but retrievals of recent data that are protected under an _**Exclusive Access Period**_ require both _authorization_ (granted by the PI of an observing program) and authentication (i.e., by logging in to the Portal, or by using a _**Auth.MAST token**_). See [MAST Accounts](https://outerspace.stsci.edu/spaces/MASTDOCS/pages/136481910/MAST+Accounts) for details.
