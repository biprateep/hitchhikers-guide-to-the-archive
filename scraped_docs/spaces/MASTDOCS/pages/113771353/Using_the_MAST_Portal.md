The [Portal Guide](https://outerspace.stsci.edu/spaces/MASTDOCS/pages/94962825/Portal+Guide) provides comprehensive documentation for searching and retrieving data from the multi-mission archive using the rich features of the [Portal web application](https://mast.stsci.edu/portal/Mashup/Clients/Mast/Portal.html). Read in particular the chapter [Introduction to the Portal](https://outerspace.stsci.edu/spaces/MASTDOCS/pages/94962826/Introduction+to+the+Portal) for essential things to know before using the Portal, and read the [Field Guide to the Portal](https://outerspace.stsci.edu/display/MASTDOCS/Field+Guide+to+the+Portal) to familiarize yourself with the visual interface.
This chapter of the _JWST Archive Manual_ focuses on special searches or considerations that are specific to JWST data.
