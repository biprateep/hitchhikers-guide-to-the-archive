The mission search interfaces provide both a graphical user-interface (web form) and programmatic access ([via an API](https://mast.stsci.edu/search/docs/)) to mission-specific data. These interfaces allow for searches with a large set of criteria, which are customized for each mission or collection.

Info on data products
Want to learn about the products in the search form, rather than the search form itself? Visit the [MAST Data Collection Overview](https://outerspace.stsci.edu/display/MASTDATA/MAST+Data+Collection+Overview).  
NOTE: If the video is blurry when watching in full screen mode, consider changing the playback setting to a higher resolution (for example, 720p). This is accomplished using the Gear icon in the lower right. Note that the gear icon may not be present when this webpage first loads, but it will appear when the video is first played. In some cases, even if the icon says "HD" there may be a need to select the 1080p HD setting using the Gear icon to ensure the highest quality video.

Search interfaces are available for the following collections:
* [HST](https://mast.stsci.edu/search/ui/#/hst/)
* [JWST](https://mast.stsci.edu/search/ui/#/jwst/)
* [Classy](https://mast.stsci.edu/search/ui/#/classy/) (High-level Science Product collection)
* [Ullyses](https://mast.stsci.edu/search/ui/#/ullyses/) (High-level Science Product collection)  
Interfaces for more missions and collections are planned.
