<!-- Metadata: {"Header 1": "TIKE Science Platform"} -->
The TIKE platform is designed for the analysis of data from NASA's TESS and Kepler/K2 missions and features pre-installed Python software such as:
* Core scientific packages: numpy, scipy, matplotlib, pandas.
* Core astronomy packages: astropy, astroquery, pyvo.
* Data analysis packages: emcee, george, celerite.
* TESS-focused packages: lightkurve, astrocut.
* Machine learning: tensorflow, scikit-learn.
* Cloud tools: awscli, boto3.
