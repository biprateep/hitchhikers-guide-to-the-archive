2026-02-10 | [roman_0046.pmap](https://roman-crds.stsci.edu/context_table/roman_0046.pmap) | latest | Affected modes - WFI, imaging or WIM and spectral or WSM modes, darkdecaysignal, detectorstatus, and integralnonlinearity reference files. This delivery includes three new WFI reference file types for Build 21 - detectorstatus, darkdecaysignal, and integralnonlinearity. The new detector status file is a true false table of which detectors can undergo prompt data processing. The dark decay reference file has a table for every detector of a parameterized correction of a signal decaying in time with amplitude and time scale. The integral non linearity reference files are per detector and have corrections for each amplifier for sets of 128 pixels to account for analog to digital conversion. Delivering the first WFI darkdecaysignal, detectorstatus, and integralnonlinearity rmaps to initialize these new reference types and prepare for subsequent darkdecaysignal, detectorstatus, and integralnonlinearity reference file deliveries.
