Welcome to the Roman Integration & Test Archive (RITA). RITA contains data from multiple ground test campaigns for the Nancy Grace Roman Space Telescope. Data will be available for download in early 2025. Downloads from RITA are restricted to authorized accounts. Please log in to download files. Searches will return up to 5000 results.
