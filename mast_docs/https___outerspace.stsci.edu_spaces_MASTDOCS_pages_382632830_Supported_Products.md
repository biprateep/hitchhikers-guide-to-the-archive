# Known Supported Products
Jdaviz generally supports loading any kind of image or spectral data products by filename, MAST URI, numpy array object, or specutils.Spectrum object. Jdaviz attempts to identify and load spectral data using the [specutils](https://specutils.readthedocs.io/en/latest/index.html) data loaders, otherwise it uses internal logic to try and load the file. The table below lists a set of data products by collection that are known to be loadable into Jdaviz and Lcviz. However, this does not mean other data products cannot be loaded. 
Collection | Data Products  
---|---  
JWST | i2d, x1d, c1d, s2d, s3d  
HST | drz, sx1, sx2  
SDSS |  manga: LOGCUBE apogee: apStar, apVisit, aspcapStar eboss: spec-full, spec-lite   
TESS | lc  
Kepler | llc, slc  
K2 | llc, slc
