# JWST-Specific Search Components
This page describes elements of the search form which are either unique to JWST, or are shared with other missions but contain options that are best explained on a per-mission basis.
Duplication Checking
Do not attempt to check for duplicate observations using the new JWST search form; planned but unexecuted observations will not be included. You should either use the [MAST Portal](https://mast.stsci.edu) or follow the instructions in the [JWST Duplication Checking Notebook](https://spacetelescope.github.io/mast_notebooks/notebooks/JWST/duplication_checking/duplication_checking.html).
  

## Dataset ID
![](https://outerspace.stsci.edu/download/thumbnails/217350629/jwst-dataset-id.png?version=1&modificationDate=1698759267918&api=v2)
A **Dataset ID** marks a "top-level" collection of files from an Observation. The naming convention follows a truncated version of the full [JWST file naming scheme](https://jwst-pipeline.readthedocs.io/en/latest/jwst/data_products/file_naming.html); each filename in the dataset will begin with the dataset ID, followed by indicators for the detector used and product type.
* * *
## Data Type
![](https://outerspace.stsci.edu/download/attachments/217350629/jwst-data-types.png?version=1&modificationDate=1698759267687&api=v2)
The **Data Type** refers to the semantic content of the highest-level [pipeline product](https://jwst-docs.stsci.edu/accessing-jwst-data/jwst-science-data-overview) for the type of exposure that produced the data. Note that slitless and IFU spectral modes ultimately produce data of type _Spectrum_.
_Other_ data includes internal calibration and guide-star data.
* * *
## Instrument Selector
![](https://outerspace.stsci.edu/download/attachments/217350629/jw-inst-select.png?version=1&modificationDate=1698759267748&api=v2)
By default, all **Instruments** are selected. The upper right of the field presents a _Select all_ vs.  _Deselect all_ choice. You can also select individual fields.
* * *
## Moving Target Flag
![](https://outerspace.stsci.edu/download/thumbnails/217350629/jwst-mt.png?version=1&modificationDate=1698759267774&api=v2)
Enabling the **Moving Targets** flag will only return targets without fixed coordinates (i.e. solar system targets).
* * *
## Observation Mode
![](https://outerspace.stsci.edu/download/thumbnails/217350629/jwst-obs-mode.png?version=1&modificationDate=1698759267948&api=v2)
The **Observation Mode** selector is empty by default. Note that searching for "all" and "none" are equivalent, as all observing modes will be included in both cases. This mode does not include type-ahead, so you must select your choice from the drop-down menu.
* * *
## Observation and visit Numbers
![](https://outerspace.stsci.edu/download/attachments/217350629/jwst-obsnum.png?version=1&modificationDate=1698759267890&api=v2)
![](https://outerspace.stsci.edu/download/attachments/217350629/jwst-visitnum.png?version=1&modificationDate=1698759267865&api=v2)
The **Observation Number** and **Visit Number** correspond to planning information from the submitted [APT file](https://www.stsci.edu/scientific-community/software/astronomers-proposal-tool-apt). You can find the APT file, as well as a PDF summary, by searching for the corresponding proposal on the [JWST Program Information search form](https://www.stsci.edu/jwst/science-execution/program-information).
* * *
## Proposal Cycle
![](https://outerspace.stsci.edu/download/thumbnails/217350629/jwst-prop-cycle.png?version=1&modificationDate=1698759267806&api=v2)
  

**Proposal Cycle** is the number of the JWST Proposal Cycle in which the program was submitted and approved. This does not necessarily correspond to the epoch of the [scheduled observation](https://www.stsci.edu/jwst/science-execution/observing-schedules).
* * *
## Proposal Type
![](https://outerspace.stsci.edu/download/thumbnails/217350629/jwst-prop-type.png?version=1&modificationDate=1698759267835&api=v2)
**Proposal Type** is the programmatic category assigned to the proposal. There are several [Categories of JWST Science Observations](https://www.stsci.edu/jwst/science-execution/approved-programs).
