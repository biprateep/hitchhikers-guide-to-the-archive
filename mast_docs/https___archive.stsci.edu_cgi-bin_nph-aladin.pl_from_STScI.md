## Overview
_Aladin_ is an interactive sky atlas allowing the user to visualize digitized astronomical images or full surveys, superimpose entries from astronomical catalogues or databases, and interactively access related data and information from the _Simbad database_ , the _VizieR service_ and other archives for all known astronomical objects in the field.

The _Aladin sky atlas_ is available in two modes: 
  * _Aladin Desktop_ , a regular Desktop application 
  * _Aladin Lite_ a Javascript Web application. 


# Aladin Desktop
[Aladin Desktop](http://aladin.cds.unistra.fr/AladinDesktop/) is a widely-used java tool capable of addressing challenges such as locating data of interest, accessing and exploring distributed datasets, visualizing multi-wavelength data. Compliance with existing or emerging [VO](http://www.ivoa.net/) standards, interconnection with other visualisation or analysis tools, ability to easily compare heterogeneous data are key topics allowing Aladin to be a powerful data exploration and integration tool as well as a science enabler. Aladin Desktop is based on Java techonology. It requires a classical installation on the user machine. [](http://aladin.cds.unistra.fr/AladinDesktop)
[More about Aladin Desktop](http://aladin.cds.unistra.fr/AladinDesktop/)
# Aladin Lite
[Aladin Lite](http://aladin.cds.unistra.fr/AladinLite/) is a lightweight version of the Aladin tool, running in the browser and geared towards simple visualization of a sky region. Aladin Lite is powered by the WebGL technology, currently supported by any modern browser. Aladin Lite is easily [embeddable on any web page](http://aladin.cds.unistra.fr/AladinLite/doc#embedding) and can also be controlled through a [Javacript API](http://aladin.cds.unistra.fr/AladinLite/doc#API).
[Access Aladin Lite previewer](http://aladin.cds.unistra.fr/AladinLite/)
  

We also offer an [experimental viewer to visualize planetary surfaces](https://aladin.cds.unistra.fr/AladinLite/planets-explorer/ "Aladin Lite planets explorer") HiPS datasets
If you are you a developer interested in integrating Aladin Lite in your project, have a look at the [dedicated documentation](http://aladin.cds.unistra.fr/AladinLite/doc/).   
  

Aladin Lite is also available as a widget for Python notebooks: ipyaladin.
[Access ipyaladin documentation](http://aladin.cds.unistra.fr/AladinLite/ipyaladin/)
### Available data
_Aladin Desktop & Aladin Lite_ use reprocessed data based on [HiPS](http://aladin.cds.unistra.fr/hips) technology (Hierarchical Progressive Survey) with the capability to zoom and pan on any regions of the surveys. There are about [1100+ HiPS surveys](http://aladin.cds.unistra.fr/hips/list) available for 900TB of pixels provided by several collaborative servers. 
_Aladin Desktop_ provides direct data access to most of the astronomical servers over the world ([CDS](https://cds.unistra.fr/), [NED](http://ned.ipac.caltech.edu/), [ESO](http://archive.eso.org/dss/dss), [CADC](http://www.cadc-ccda.hia-iha.nrc-cnrc.gc.ca), [MAST](http://archive.stsci.edu), [HEASARC](http://skyview.gsfc.nasa.gov), [NRAO](http://www.cv.nrao.edu), [ROE](http://www-wfau.roe.ac.uk), [IMCCE](http://vo.imcce.fr), etc), and obviously, local user data.
