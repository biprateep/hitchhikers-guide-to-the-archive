### Summary
This page contains bulk download scripts to retrieve sets of K2 data. The bulk scripts use [cURL](https://curl.haxx.se) to download files of the given type, one at a time, from MAST. Specific lines in each bulk download script can be run separately to retrieve single files, or subsets of files, as desired.
