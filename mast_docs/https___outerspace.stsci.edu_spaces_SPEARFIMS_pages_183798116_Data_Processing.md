#  [Data Processing](https://outerspace.stsci.edu/spaces/SPEARFIMS/pages/183798116/Data+Processing)
  * Created by  ![User icon: alucy](https://outerspace.stsci.edu/s/aneund/9204/1jtwilc/_/images/icons/profilepics/anonymous.svg) [Adrian Lucy](https://outerspace.stsci.edu/display/~alucy%0A), last updated on [Apr 04, 2023](https://outerspace.stsci.edu/pages/diffpagesbyversion.action?pageId=183798116&selectedPageVersions=15&selectedPageVersions=16 "Show changes") 1 minute read 


This chapter describes the data processing which transformed the raw telemetry into final data products. The processing steps are presented in the order they were performed. This overview is not comprehensive; please contact us if there is information you need that is not presented here.
_Note: the two mission teams,[FIMS and SPEAR](https://outerspace.stsci.edu/spaces/SPEARFIMS/pages/183798104/Mission+Teams), processed the data independently during certain steps. In cases where the processing pipelines differ significantly, the articles in this chapter will note the difference where it occurs. See [FIMS vs. SPEAR Features](https://outerspace.stsci.edu/spaces/SPEARFIMS/pages/183798179/FIMS+vs.+SPEAR+Features) for an overview of the most important differences.
