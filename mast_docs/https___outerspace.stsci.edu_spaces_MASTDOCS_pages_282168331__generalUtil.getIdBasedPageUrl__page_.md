The Calibration and Supplemental Search Interface (CaSSI) provides access to uncommonly accessed and restricted [JWST](https://mast.stsci.edu/cassi/#/jwst) and [Roman](https://mast.stsci.edu/cassi/#/roman) mission product categories. The products include pre-launch test data, telemetry and, for Roman, calibration reference products. CaSSI also offers an API for search and data retrieval.

## Limitations
Be aware that the CaSSI web interfaces will return no more than 5000 results. If the reported number of matches in the results table is exactly 5000 it likely means that not all matching archived products are displayed. Repeat the search with more restrictive search criteria to keep the results to within the limit. An alternative is to retrieve a curl script via the **Show API Query** button, remove the limit from the script, and execute the search from the command line. 
Some data groups, such as supplemental JWST data, include millions of files. If you execute an API search with increased or missing limits, the search may return very large numbers of results. Such searches may take many minutes to execute, or may time out on the server. It is best to start with highly restrictive search criteria, or to first perform a count of the number of results, before performing an extensive or unconstrained query.
## Authorization
All except the calibration product categories available in CaSSI require per-user authorization. Please login to CaSSI to retrieve data. If you need, but do not have, authorization contact the MAST [Archive Support](https://outerspace.stsci.edu/spaces/MASTDOCS/pages/94963058/Archive+Support) to request it, mention the mission for which you need authorization, and describe the functional reason for the request.
