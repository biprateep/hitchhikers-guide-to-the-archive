# Browsing and Visualization
The [MAST Portal](https://mast.stsci.edu/portal/Mashup/Clients/Mast/Portal.html) provides functionality to assess the applicability of search results to your scientific purpose. See the [Portal Guide](https://outerspace.stsci.edu/spaces/MASTDOCS/pages/94962825/Portal+Guide) for details, including descriptions of the following: 
  * See [Browsing Data](https://outerspace.stsci.edu/spaces/MASTDOCS/pages/94962939/Browsing+Data) for a high-level view of how to navigate between multiple searches.
  * See the [Search Results Grid](https://outerspace.stsci.edu/spaces/MASTDOCS/pages/94962944/Search+Results+Grid) for tools that help you browse observation metadata, or capture the results of a query as a persistent URL.
  * See [Refining Results with Filters](https://outerspace.stsci.edu/spaces/MASTDOCS/pages/94962974/Refining+Results+with+Filters) to down-select which results are presented.


Tools for browsing the content of data files are available via links embedded in each row of the table. Only tools that are applicable for the data type(s) have active buttons or links. See the following articles in the [Portal Guide](https://outerspace.stsci.edu/spaces/MASTDOCS/pages/94962825/Portal+Guide) for extensive descriptions:
  * See [Data Browsing Tools](https://outerspace.stsci.edu/spaces/MASTDOCS/pages/94962980/Data+Browsing+Tools) for options to display image thumbnails or cutouts, overlay an image on a sky survey, or view timeseries data.
  * See the [Astroview](https://outerspace.stsci.edu/spaces/MASTDOCS/pages/94962998/AstroView) tool for displaying wireframes of search results superimposed on a sky survey


The primary aim of data visualization in the [MAST Portal](https://mast.stsci.edu/portal/Mashup/Clients/Mast/Portal.html) is to enable the discovery and selection of data that are relevant and of sufficient quality and depth to address the searcher's scientific goals. This is particularly important for spectroscopic data. For JWST, which produces very advanced data products, a new, more capable tool takes the place of the classic spectral viewer tool.   

The [Jdaviz](https://jdaviz.readthedocs.io/en/latest/) package enables quick-look and analysis in your browser of JWST Level-3 spectroscopic data products. Follow this tutorial to learn how to view the default Minimum Recommended Product for an observation. For more detailed documentation, see the complete [Jdaviz in MAST](https://outerspace.stsci.edu/spaces/MASTDOCS/pages/150113048/Jdaviz+in+the+MAST+Portal) guide. The following example illustrates how to invoke the Jdaviz tool in the MAST Portal.
  
| Instruction | Notes  
---|---|---  
1 |  Perform a Portal search by target or coordinates. A [basic](https://outerspace.stsci.edu/spaces/MASTDOCS/pages/94962886/Basic+Search) or [advanced](https://outerspace.stsci.edu/spaces/MASTDOCS/pages/94962889/Advanced+Search) search work equally well. See the [Jdaviz Portal Search](https://outerspace.stsci.edu/display/MASTDOCS/Searching+for+Compatible+Data) page for more detailed information on navigation from the Portal basic and SI keyword search pages. | ![](https://outerspace.stsci.edu/download/thumbnails/113771401/menu_selectCollection_obs.png?version=1&modificationDate=1664284153751&api=v2)  
2 | A **Jda** action icon will appear for rows in the results table that are viewable with the Jdaviz tool. | ![Search rows displaying the Jdaviz icons](https://outerspace.stsci.edu/download/thumbnails/113771401/mast_jdaviz_new_portal_icon.png?version=1&modificationDate=1664284153858&api=v2)Search rows displaying the Jdaviz icons  
3 |  Click the **Jda** action icon to reveal a dropdown menu, with multiple options. Select the first (default) option, which will display the default Minimum Recommended Product in Jdaviz.  For information for the option __**Select which file to open**__ , see [Searching for Jdaviz-Compatible Data](https://outerspace.stsci.edu/display/MASTDOCS/Searching+for+Compatible+Data).  | ![](https://outerspace.stsci.edu/download/attachments/113771401/menu_jdaviz_pid1128.png?version=1&modificationDate=1664284153710&api=v2)  
4 |  The "**Search through other data products related to this observation** " option will open a new browser tab and load the designated data product into the "Jdaviz in MAST" web application. The result will look something like the figure below. This specific example shows the file **jw01128-o007_s00002_nirspec_f070lp-g140h-s200a2-subs200a2_x1d.fits** ![](https://outerspace.stsci.edu/download/attachments/113771401/panel_jdaviz_pid1128.png?version=1&modificationDate=1664284153365&api=v2)  
Once loaded, the Jdaviz in MAST spectroscopic quicklook page can be used to quickly inspect a spectroscopic data product. On the left-hand side is a sidebar panel of information. In the center/right the data product is loaded into the Jdaviz application. Jdaviz has different configurations which are optimized for different types of data products, i.e. 1-D, 2-D, or 3-D spectroscopic data. The example shown above has loaded data into the [Specviz](https://jdaviz.readthedocs.io/en/latest/specviz/index.html) configuration. For a complete description of the quick-look page and the available Jdaviz configurations, see the Viewing Individual Spectra article (not yet available) for more details
