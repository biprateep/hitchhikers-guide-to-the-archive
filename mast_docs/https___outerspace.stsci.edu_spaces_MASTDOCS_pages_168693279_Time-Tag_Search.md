# Selecting an Operating Mode
The Hubble Space Telescope's instruments have many operating modes; we can use the 'Operating Mode' condition allows us to search for a particular mode. In this case, we want to filter for observations taken in the ['time-tagged' mode](https://hst-docs.stsci.edu/cosdhb/chapter-5-cos-data-analysis/5-4-working-with-time-tag-data).
![](https://outerspace.stsci.edu/download/attachments/168693279/classic_user_field.png?version=1&modificationDate=1665597307758&api=v2)
In the classic search form, 'Operating Mode' could be found as an option under 'User-specified' field. You could then enter your desired criteria in the blank box to the right.
![](https://outerspace.stsci.edu/download/attachments/168693279/operating_mode.gif?version=1&modificationDate=1665680996387&api=v2)
Although the drop-down menu is still available, the new form also supports type-ahead. This allows you to type the name of the field, or a part of its description, and matching results will be returned.
* * *
# Limiting Results to a Filter
In this search, we also choose to limit results that used filter G230L. For more filters, see the [HST Filters/Gratings](https://archive.stsci.edu/hst/filterlist.html?style=Classic) Page.
![](https://outerspace.stsci.edu/download/attachments/168693279/classic_filters.png?version=1&modificationDate=1665609904299&api=v2)
The 'Filters/Gratings' field provided a space to input your desired filters, although until the search was executed, you could not verify the validity of the filter.
![](https://outerspace.stsci.edu/download/attachments/168693279/new_filters.gif?version=1&modificationDate=1665610172266&api=v2)
In the new search form, the 'Filters/Gratings' field supports type-ahead; this makes it easier to find and enter a valid value.
* * *
# Exporting Results
![](https://outerspace.stsci.edu/download/thumbnails/168693279/export_table.png?version=1&modificationDate=1665685704093&api=v2)
This search returns 1455 results; you may wish to apply additional filters, or export the results to a file. Exporting is a new feature that allows you to turn the results table into a CSV, JSON, or [VO table](https://archive.stsci.edu/virtual-observatory). After clicking on 'Export Table', all three file types will be displayed on options. Clicking on one of these options will begin the download.
