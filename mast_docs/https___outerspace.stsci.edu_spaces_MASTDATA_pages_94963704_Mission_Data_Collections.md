# MAST Interfaces
MAST provides a variety of interfaces to access its hosted data collections. Tables in this document provide a few-letter code to indicate which interface(s) may be used to access data from a given collection. The following table lists the codes and their meaning. 
Interface Code | Description  
---|---  
Interface Code | Description  
CJ |  [CasJobs](http://mastweb.stsci.edu/mcasjobs/) (requires a separate account)  
C.M | [Catalogs.MAST](https://catalogs.mast.stsci.edu/)  
D | Direct download (usually from a link on a MAST webpage)  
E.M | [Exo.MAST](https://exo.mast.stsci.edu/)  
G | HTTP Get Request  
P |  [Portal](https://mast.stsci.edu/portal/Mashup/Clients/Mast/Portal.html) user interface  
PX |  [Portal](https://mast.stsci.edu/portal/Mashup/Clients/Mast/Portal.html) Cross-match  
VOT | Virtual Observatory Tables interface  
WF | Custom MAST web form
