# Component Overview
This section describes the Search Bar, which is available at the bottom of the Search Form.
## Component Features and Functionality
The Search Bar consists of three buttons: '**CLEAR FORM** ', '**SEARCH** ', and '**SHOW API QUERY** '.
* * *
![](https://outerspace.stsci.edu/download/thumbnails/104958267/thesearchbar_clearform.png?version=1&modificationDate=1629837494510&api=v2)
The "CLEAR FORM" Button
**'CLEAR FORM'** resets the form to its default values: all changes made to the form are removed. Use this button to start a new search.
![](https://outerspace.stsci.edu/download/thumbnails/104958267/thesearchbar_search.png?version=1&modificationDate=1629837494472&api=v2)
The "SEARCH" Button
**'SEARCH'** submits the query as entered in the search form and shows the results in the [Search Results Table](https://outerspace.stsci.edu/spaces/MASTDOCS/pages/107840287/Search+Results+Page).
![](https://outerspace.stsci.edu/download/thumbnails/104958267/thesearchbar_showapiquery.png?version=1&modificationDate=1629837494435&api=v2)
The "SHOW API QUERY" Button
'**SHOW API QUERY'** opens an overlay with a written, programmatic-based query, which can be used in a terminal or included in programming scripts. The API queries shown in the overlay match the search constraints specified in the search form.
